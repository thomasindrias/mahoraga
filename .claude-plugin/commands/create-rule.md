---
description: Scaffold a custom Mahoraga detection rule
allowed-tools: Bash(npx:*)
---

Run `npx mahoraga-cli create-rule` to start the interactive rule scaffolding wizard.

This is an interactive command — it will prompt for the rule name, event types, and other options.

After the command completes, summarize:
- Generated rule file location
- Generated test file location
- Next steps for implementing the detection logic

Suggest the user invoke the **mahoraga-custom-rules** skill for guidance on implementing the rule.
